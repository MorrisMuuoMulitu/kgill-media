## Development Workflow

### Prerequisites
- Node.js 16+
- npm or yarn package manager
- Git for version control

### Setup
```bash
# Clone repository
git clone [repository-url]
cd kgill-media

# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production
```bash
# Create production build
npm run build

# Preview production build
npm run preview
```

## Deployment Options

The website can be deployed to:
- **Netlify**: Zero-config deployment with automatic optimizations
- **Vercel**: Edge network deployment with smart caching
- **GitHub Pages**: Free hosting with automatic SSL
- **Traditional Hosting**: Upload dist folder to any web server

See `DEPLOYMENT_GUIDE.md` for detailed instructions.
